Connect MCP Inspector to SSE Server

Click the link below to connect the Inspector to your SSE server on port 8000:

Connect to SSE Server (port 8000)

Or use the Inspector directly in browser:

Note: The MCP Inspector should connect automatically when you click the link above.

The SSE server is running at http://127.0.0.1:8000/sse

Test with curl:

# 1. Connect to SSE endpoint to get session
curl -N -H "Accept: text/event-stream" http://127.0.0.1:8000/sse

# You'll get a session_id, then use it to send messages to:
# http://127.0.0.1:8000/messages/?session_id=YOUR_SESSION_ID

Connection Details:

  • Server URL: http://127.0.0.1:8000/sse
  • Transport: SSE (Server-Sent Events)
  • Protocol: MCP (Model Context Protocol)
